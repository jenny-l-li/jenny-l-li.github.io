WEB APPLICATION

LIVE SITE      GITHUB

Created website that displays a database of songs using HTML/CSS, JavaScript, Handlebars.js, and node.js.

DESCRIPTION

Website stores songs in a database for user to access. The user can click on the image of the song to view and listen to any song in the database.

FUNCTIONALITY

  • Displays all songs and their metadata, including song title, artist name, album name, and song or album artwork
  • Play and pause song using an audio player
  • Add new song to database
  • Delete song from database
  • Display albums only
